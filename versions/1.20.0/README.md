# Enthusia AutoClicker for Minecraft 1.20.0

This standalone Gradle build produces a Fabric jar for Minecraft 1.20.0.

## Requirements

- Java 17
- Fabric: Fabric Loader and Fabric API
- Fabric configuration screen integration: Mod Menu is recommended

## Build

Run `gradlew.bat build` on Windows or `./gradlew build` on Linux and macOS.

The loader-specific jar is written to `fabric/build/libs`.
